# Formulario Construcao

Frontend estatico do formulario para empresarios.

## Arquivos

- `index.html`
- `config.js`
- `cosmob pequeno.png`
- `netlify.toml`

## Deploy na Netlify

1. Crie um repositorio novo no GitHub para esta pasta.
2. Importe o repositorio na Netlify.
3. Deixe `Base directory`, `Publish directory` e `Build command` vazios.
4. O site vai publicar a raiz do repositorio.

## Backend

Por padrao, `config.js` aponta para:

- `https://backend-production-5b5cc.up.railway.app`

Se a URL do backend mudar, edite `config.js`.
